WHAT?

OpsTheatre is a modular app with a backend built in node.js. It's frontend is currently an angular app which talks to the backend through REST API's. Both the frontend and the backend support plugins or modules

WHY?

At OlinData we do a lot of consulting for different clients. Each client uses some combination of dashboards for operations. Nagios, Puppet Enterprise, The Foreman, Kibana, etc. While things like OpenID and SSO are nice, they still don't prevent you from having to be logged into multiple different dashboards just to be able to manage a single infrastructure.

OpsTheatre aims to solve that problem by implementing a standard API: OpsAPI. Any dashboard that supports OpsAPI can be easily hooked up to OpsTheatre. However, we want this to be widely adopted, so we encourage other dashboards to also support this API so we can standardise things across the board.
